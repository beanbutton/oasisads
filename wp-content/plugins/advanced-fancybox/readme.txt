== Description ==
Fancybox (lightbox) fully compatible for wordpress. You can add (so far):
<ul>
	<li>Image link (local or external)</li>
	<li>Html content (or normal - Content hast to be added using a custom field with normal text or html content)</li>
	<li>Link to another webpage (webpage content will show in iframe)</li>
</ul>
Requires JavaScript Enabled. Uses jQuery (Fancybox and TinyMce Editor).<br/>
<b>You can modify all settings fancybox code allows you to modify!</b>

== Installation ==

Upload the Advanced Fancybox plugin to your blog(in the wp-content/plugins directory) (be sure to upload the "advanced-fancybox" directory or it  will not work.

Activate it in the plugins menu in your wp-admin control panel and voila! you're done!

== Frequently Asked Questions ==

= How do i use custom fields? =

If you want html or normal text content in your fancybox, you must use custom fields.<br/>
The custom field content can be any but the name must be one of the following:
<ul> 
	<li>fancybox</li> 
	<li>fancybox1</li> 
	<li>fancybox2</li> 
	<li>fancybox3</li> 
</ul>
